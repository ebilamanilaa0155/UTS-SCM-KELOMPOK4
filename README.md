Aplikasi Kalkulator Sederhana
Deskripsi
Aplikasi ini adalah kalkulator sederhana berbasis antarmuka grafis (GUI) yang memungkinkan pengguna melakukan operasi aritmatika dasar dan operasi aritmatika tingkat lanjut dengan mudah. Selain itu, aplikasi ini juga dilengkapi dengan fitur riwayat perhitungan sehingga pengguna dapat melihat kembali hasil perhitungan sebelumnya.
Tampilan aplikasi dirancang minimalis dan responsif, dengan pembagian antara panel kalkulator dan panel riwayat untuk meningkatkan kenyamanan penggunaan.

Fitur
1. Operasi aritmatika dasar:
  -Penjumlahan (+)
  -Pengurangan (-)
  -Perkalian (*)
  -Pembagian (/)
2. Operasi aritmatika tingkat lanjut
  -akar  (√)
  -Prima (P)
3. Input angka 0–9 dan desimal (.)
4. Tombol C (Clear) untuk menghapus input
5. Tombol hapus (backspace) untuk menghapus satu karakter
6. Tombol = untuk menampilkan hasil
7. Riwayat perhitungan:
  -Menyimpan hasil perhitungan sebelumnya
  -Menampilkan daftar riwayat
8. Tombol Hapus Semua untuk mengosongkan riwayat

Cara Menjalankan Aplikasi
1. Clone Repository
git clone https://github.com/ebilamanilaa0155/UTS-SCM-KELOMPOK4.git
cd UTS-SCM-KELOMPOK4
2. Jalankan Aplikasi

Cara Penggunaan
1. Masukkan angka menggunakan tombol yang tersedia.
2. Pilih operator matematika (+, -, *, /).
3. Tekan tombol = untuk melihat hasil.
4. Apabila ingin menggunakan Operasi matematika tingkat lanjut  (√) dan (P) tidak perlu menggunakan tombol =
5. Gunakan tombol:
  -C untuk menghapus semua input
  -⌫ untuk menghapus satu digit
6. Lihat riwayat perhitungan di panel kanan.
7. Klik Hapus Semua untuk menghapus seluruh riwayat.
