Aplikasi Kalkulator Sederhana
Deskripsi
Aplikasi ini adalah kalkulator sederhana berbasis antarmuka grafis (GUI) yang memungkinkan pengguna melakukan operasi aritmatika dasar dengan mudah. Selain itu, aplikasi ini juga dilengkapi dengan fitur riwayat perhitungan sehingga pengguna dapat melihat kembali hasil perhitungan sebelumnya.
Tampilan aplikasi dirancang minimalis dan responsif, dengan pembagian antara panel kalkulator dan panel riwayat untuk meningkatkan kenyamanan penggunaan.

Fitur
1. Operasi aritmatika dasar:
  -Penjumlahan (+)
  -Pengurangan (-)
  -Perkalian (*)
  -Pembagian (/)
2. Input angka 0–9 dan desimal (.)
3. Tombol C (Clear) untuk menghapus input
4. Tombol hapus (backspace) untuk menghapus satu karakter
5. Tombol = untuk menampilkan hasil
6. Riwayat perhitungan:
  -Menyimpan hasil perhitungan sebelumnya
  -Menampilkan daftar riwayat
7. Tombol Hapus Semua untuk mengosongkan riwayat

Cara Menjalankan Aplikasi
1. Clone Repository
git clone https://github.com/ebilamanilaa0155/UTS-SCM-KELOMPOK4.git
cd UTS-SCM-KELOMPOK4
2. Jalankan Aplikasi

Cara Penggunaan
1. Masukkan angka menggunakan tombol yang tersedia.
2. Pilih operator matematika (+, -, *, /).
3. Tekan tombol = untuk melihat hasil.
4. Gunakan tombol:
  -C untuk menghapus semua input
  -⌫ untuk menghapus satu digit
5. Lihat riwayat perhitungan di panel kanan.
6. Klik Hapus Semua untuk menghapus seluruh riwayat.
